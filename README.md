# 🤖 CYBORG X — Future of Human-Machine Evolution

A futuristic, fully responsive cyberpunk-themed landing page featuring dark neon aesthetics, interactive particle systems, glitch animations, glassmorphism UI, an animated SVG cyborg head with real-time HUD readouts, and a complete contact form — all built with zero external dependencies using pure HTML5, CSS3, and Vanilla JavaScript.

---

## ✨ Features

CYBORG X is a complete, production-ready landing page that demonstrates modern front-end development techniques without relying on any external frameworks or libraries. The project showcases advanced CSS animations, canvas-based particle systems, SVG artwork with embedded animations, responsive design across all device sizes, and accessible markup following web standards.

The entire website runs as a static site with just three core files — index.html, styles.css, and main.js — making it extremely lightweight (under 50KB total), fast to load, and easy to deploy on any hosting platform including GitHub Pages, Netlify, Vercel, or any basic web server.

Every visual effect from the floating cyborg head to the glitch text to the particle network is built from scratch, proving that stunning web experiences can be created with fundamental web technologies alone.

## 🛠️ Tech Stack

| Technology | Version | Purpose |
|------------|---------|---------|
| HTML5 | 5 | Semantic page structure and content |
| CSS3 | 3 | Styling, animations, responsive design, glassmorphism, custom properties |
| JavaScript | ES6 | Particles, cursor, typing, scroll animations, form validation, HUD simulation |
| Google Fonts | CDN | Orbitron (display), Rajdhani (body), Share Tech Mono (monospace) |
| Canvas API | Native | Particle system rendering with 2D context |
| IntersectionObserver API | Native | Scroll-triggered reveal animations and counter triggers |
| SVG | Inline | Cyborg head visual, icons, gauge rings, logo |

## 🚀 How to Run on Your Local System

### Option A — Python Local Server

```bash
# Step 1: Clone the repository
git clone https://github.com/your-username/cyborg-landing.git

# Step 2: Navigate into the project folder
cd cyborg-landing

# Step 3: Start the local server
python -m http.server 8000

# Step 4: Open your browser and go to
# http://localhost:8000
```

---

### Option B — VS Code Live Server

```
Step 1: Open VS Code
Step 2: Go to Extensions (Ctrl + Shift + X)
Step 3: Search for "Live Server" by Ritwick Dey
Step 4: Click Install
Step 5: Open the cyborg-landing folder in VS Code (File → Open Folder)
Step 6: Open index.html
Step 7: Click "Go Live" button at the bottom right corner
Step 8: Browser opens automatically with the website
```

---

### Option C — Direct Browser Open (No Server)

```
Step 1: Navigate to the cyborg-landing folder on your computer
Step 2: Double-click on index.html
Step 3: The website opens in your default browser
```

> **Note:** This method works but Google Fonts may load slower compared to running a local server.
